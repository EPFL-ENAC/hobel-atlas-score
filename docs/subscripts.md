# Field Name Formatting with Subscripts

### Method 1: Underscore Notation

- `CO_2` → CO₂
- `PM_2.5` → PM₂.₅
- `H_2O` → H₂O

### Method 2: Curly Brace Notation (Advanced)

For more complex subscripts, you can use LaTeX-like syntax:

- `CO_{2}` → CO₂
- `PM_{2.5}` → PM₂.₅
- `C_{6}H_{12}O_{6}` → C₆H₁₂O₆
